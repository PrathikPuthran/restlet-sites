What to know, what new features are coming up next and what were the latest enhancement released on the platform? This page will contain up-to-date information. In addition, make sure to check out blog often at http://blog.restlet.com.

Limited public beta (02/2013)

  • Local entity stores, based on Apache Cassandra
  • Local file stores, based on AWS S3
  • API creation, hosting, management and usage
  • Basic API promotion in the Catalog area

Extended public beta (03/2013)

  • Full cell deletion (freeing the subdomain name)
  • Revert 'Published' APIs to 'Draft' possible (if has no external user)
  • Custom domain names (additional endpoints)
  • Faster API deployment times (separate downloads generation)
  • Improved overall stability (UI, deployment and runtime)

Public beta (04/2013)

  • Complex entities and representations
  • Composite APIs (local and optimized API import)
  • Enhanced mapping (source selection, basic functions)
  • External Data Store wrappers (RDBMS, spreadsheets)

Public beta (05/2013)

  • Intra-region fail-over (APIs and entity stores)
  • Better security (HTTPS, rate limitation, IP blocking)
  • Premium plans (pricing, accounting, payment)
  • Stabilization & feed-back processing

Version 1.0 (06/2013)

  • Cross-region fail-over (US West and East coasts)
  • More downloads (GWT/Android/.NET client SDKs, code export)
  • Better user documentation (user guide, tutorials, samples)
  • Commercial launch!
