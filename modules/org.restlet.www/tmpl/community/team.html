Team

Core committers

Here is the list of core developers with commit privileges over the whole source code base. They are responsible of the project management, technical architecture, design and core development. Their work is funded by Restlet S.A.S.:

  • Jérôme Louvel, founder and lead developer
  • Thierry Boileau, community manager and core developer
  • Thierry Templier, core developer

Extension committers

Some users from the Restlet community have stepped up and taken some responsibilities. Those persons are extension leads, driving a specific Restlet extension with commiter rights on the extension source code.

  • Shaun Elliott, Stephan Koops, JAX-RS extension
  • Rémi Dewitte, Lucene extension
  • Kristoffer Gronowski, Martin Svensson, OAuth extension
  • Rhett Sutphin, Spring extension
  • Brunot Harbulot, SSL extension
  • Marcelo Ochoa, XDB extension
  • Bryan Hunt, Wolfgang Werner, OSGi edition
