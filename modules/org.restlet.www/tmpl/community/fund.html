Fund the project

Introduction

When Restlet S.A.S. founded the open source Restlet project, it made the choice of Professional Open Source. It means that it contributes most of the development effort, pays the core developers and the hosting expenses, provides technical leadership and integrates contributions from the community like bug reports, patches and documentation improvements.

Professional Open Source

In order to balance this significant investment, Restlet S.A.S. has developped consulting services and technical support activities based on the success of the Restlet project. If you can afford to become a customer, this is a great way to contribute to the funding of the Restlet project.
