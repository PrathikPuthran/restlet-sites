Discussion forums

There are two main ways to get in touch with our community and ask a question or start a discussion.

Stack Overflow

Restlet Framework users have increasingly used this popular web site to post their questions and answer to each other. We encourage you to use it as it tend to provide rapide and good quality answers. The Restlet core team also tries to participate.

  • Search questions tagged 'restlet'.
  • Ask a new question, adding the 'restlet' tag to it.

Mailing lists

This is traditional way used so far by Restlet Framework users and developers to communicate. To participate, just follow these steps:

  1. Create an account on Tigris.org (or login if you have one already),
  2. Subcribe to the "Discuss" mailing list (other lists available),
  3. Post your message using:
    • a mail client,
    • the web site,
    • the newsgroup gateway.

You can also search the archives.
