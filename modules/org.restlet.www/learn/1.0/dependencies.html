Restlet 1.0 - Dependencies

  • For all applications: Java SE 5.0 or above. Java SE 1.4 also supported using Retrotranslator.
  • For standalone HTTP servers: AsyncWeb 0.8, Jetty 6.1 and Simple 3.1
  • For embedded HTTP servers: Servlet 2.4 containers like Apache Tomcat
  • For dynamic documents: JSON 2.0, FreeMarker 2.3, Velocity 1.4 and XML (via SAX or DOM)
  • For uploading large files: Apache FileUpload 1.1
  • For sending email: JavaMail 1.4
